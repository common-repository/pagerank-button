Shows your site pagerank instantly in the widget area without editing code.

== Description ==

PageRank is one of the methods Google uses to determine a page's relevance or importance. Shows your site pagerank instantly in the widget area without editing code.

This plugin send information through pagerank checker site and the image is hosted on that site. Visit http://www.checkpagerank.info/ for complete results including link popularity, pagerank validity, Alexa rank and domain age.


== Installation ==

1. Upload `pagerank-button` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets in WordPress
4. Select the section where you want the widget to appear and click Show
5. Click save changes
